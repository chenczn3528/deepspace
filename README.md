### 浏览器统计：
- 谷歌浏览器，网络好的情况下一切正常 
- edge，网络好的情况下一切正常 
- 小米浏览器，出金动画、抽卡音效加载不出来，应该是视频和音频没法同时加载

- UC浏览器，所有音频、视频只能选一个放 
- QQ浏览器，下载完视频之前都会显示不对，下载巨慢无比 
- 百度，强制播放模式
